# dashboard

**Status: belum diimplementasikan — direktori ini akan diisi pada waktu hacking babak final WRECK-IT 7.0.**

## Apa yang akan dibangun di sini

**TALOS Command Dashboard** — antarmuka web ringan untuk pusat kendali keamanan:

- **Command Overview**: ringkasan ancaman aktif, agen daring, waktu containment, feed aksi otonom real-time.
- **Incident Detail**: linimasa serangan, visualisasi blast radius, host terdampak, rencana respons otonom.
- **Fleet & Hosts**: inventaris host beserta status perlindungan/isolasi.
- **System Health**: kesehatan detection engine, cakupan agen, kebijakan otonomi, audit keputusan.

Tema gelap sebagai tema utama (kenyamanan pemantauan jangka panjang) dengan varian tema terang. Purwarupa UI high-fidelity untuk keempat layar tersebut telah dirancang dan dilampirkan pada Lampiran B proposal.
